이전 작업을 파악해서 지금 어디서부터 이어서 하면 될지 알려줘.

1. `git log --oneline -10` 으로 최근 커밋 확인
2. `git diff HEAD~1 HEAD --stat` 으로 마지막 커밋에서 바뀐 파일 확인
3. `.claude/plans/PLAN.md` 읽어서 완료된 Phase와 남은 항목 파악
4. 메모리 파일(`~/.claude/projects/.../memory/MEMORY.md`)이 있으면 읽고 관련 메모리 파일 로드
5. 아래 형식으로 요약 보고:

```
## 마지막 작업
- 커밋: [커밋 메시지]
- 변경 파일: ...

## 계획서 진행 상황
- 완료: Phase N ...
- 남은 것: ...

## 이어서 할 작업
- 다음 추천 작업: ...
```

6. 바로 이어서 작업할지 물어보고 사용자가 원하면 시작
